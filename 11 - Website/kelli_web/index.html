Summary: Latitude vs X

The purpose of this prodect was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to collect weather data, such as temperature, coudiness, humidity and windspeed, for over 500 randomly selected cities. After assembling the dataset, I used plotted each of the different weather variables vs. latitude. This website provides the source data, the visualizations that resulted from my analysis, and explanations and descriptions of any of the trends I discovered.
